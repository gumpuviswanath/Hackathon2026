# Customer Onboarding Portal

Dashboard and navigation shell for the four modules.

```powershell
cd Customer-Onboarding-Portal
npm install
npm start
```

Default URL: `http://localhost:3005`

The dashboard aggregates:

- customer-service: `http://localhost:3000/customer/all`
- account-service: `http://localhost:3002/account/all`
- product-service: `http://localhost:3003/product/all`
